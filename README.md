# 📱 Automação de Processos com Excel + VBA

Automação completa de processos em Excel utilizando VBA para criação de dashboards, relatórios e ferramentas interativas.

## ✨ Funcionalidades

- **Macros VBA**: Automação de tarefas repetitivas
- **UserForms**: Interfaces gráficas customizadas
- **Pivot Tables**: Análise dinâmica
- **Charts**: Gráficos interativos
- **API Integration**: Conexão com APIs externas
- **Data Validation**: Validação de dados
- **Performance**: Otimização de código

## 🛠️ Tecnologias

- **Excel 365**
- **VBA (Visual Basic for Applications)**
- **Power Query**: Extração de dados
- **Power Pivot**: Modelagem de dados

## 🚀 Como Usar

```
1. Abra o arquivo .xlsm
2. Alt + F11 para acessar VBA Editor
3. Execute as macros
4. Interaja com os UserForms
```

## 📁 Estrutura

```
excel-vba/
```
